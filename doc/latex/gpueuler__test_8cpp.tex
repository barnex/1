\subsection{Detailed Description}
This test program runs a small simulation using \hyperlink{structgpuconv1}{gpuconv1} and gpueuler. The initial magnetization and demag tensor are read from testm0.t and testkernel.t. A few time steps are taken and one spin of the result is compared with its known solution.
