\subsection{Detailed Description}
The params struct stores simulation parameters like
\begin{DoxyItemize}
\item material constants
\item simulation size
\item demag type
\item time stepping parameters By convention, these parameters are only changed by the user, {\itshape not\/} by the simulation itself (e.g.: a solver itself must not change the maxDt parameter, etc.).
\end{DoxyItemize}

Should there be a need for a central place to store all variables that {\itshape are\/} changed during the simulation (like m, h, time, ...), then a second struct can be made for this.

\begin{DoxyWarning}{Warning}
Unless otherwise noted, all values represent internal units. Conversion from SI units is typically done right after the input file is read, and conversion back to SI is done right before output is written. In between, everything is doen in internal units, even if not noted explicitly.
\end{DoxyWarning}
The internal program units are well adapted to the intrinsic scale of the problem. This avoids having to deal with extremely large or small quantities, which may lead to numerical errors. Our program units are chosen so that the values of the following four quantities become one:


\begin{DoxyItemize}
\item msat (saturation magnetization of the used material, A/m)
\item eaxch (exchange constant of the used material, J/m)
\item gamma0 (gyromagnetic ratio, 2.211E5 m/As)
\item mu0 (permeability of vacuum, 4pi$\ast$1E-\/7 N/A$^\wedge$2)
\end{DoxyItemize}

This leads to the following set of program units:


\begin{DoxyItemize}
\item Unit of field: msat $\ast$ mu0
\item Unit of length: sqrt(2aexch/(mu0 $\ast$ msat$^\wedge$2)), the exchange length
\item Unit of time:1/(gamma0 $\ast$ msat)
\item Unit of energy: aexch $\ast$ unitlength
\end{DoxyItemize}

The param struct provides an easy way to keep track of the internal simulation units. A new unit system is created with \hyperlink{param_8cpp_ab127140cec953de6afa605d7aa314017}{new\_\-param()}, after which aexch and msat can be set. Then, \hyperlink{param_8cpp_afef99c49ae98fd4a2b4844c484975aec}{unitlength()}, \hyperlink{param_8cpp_afd5c91ab91885614920f91c05d75e3e0}{unittime()}, \hyperlink{param_8cpp_a3d43510462f4fb3eef35ed4090b6a44d}{unitfield()} and \hyperlink{param_8cpp_aad86c3fa9ec316ed5e1462e843b66e6e}{unitenergy()} can be called to get the respective internal units expressed in SI units.

\begin{DoxyNote}{Note}
gamma0 and mu0 are set to their default values, but can be changed if desired.
\end{DoxyNote}
